3 # Observação
4 O arquivo Principal apenas roda o programa.cpp que está na pasta Sample que serve como teste para a aplicação principal
5 
6 ## Como compilar o software e o arquivo de teste
7 
8 Para compilar o software e necessário estar na pasta Source e usar o seguinte comando:
9 
10 ```
11 $ make
12 ```
13 
14 Para compilar o arquivo de teste do software é necessário rodar o seguinte comando:
15 
16 ```
17 $make test
18 ```
19 
20 ## Como rodar o programa principal
21 
22 ```
23 $make run
24 ```
25 
26 ## Como rodar os testes e analisar a cobertura
27 
28 Ainda na pasta de Source e necessário executar o seguinte código:
29 
30 ```
31 $ make run_test
32 ```
33 Para ver a cobertura de testes
34 
35 ```
36 $ make gcov
37 ```
38 
39 ## Como limpar o programa depois da execução
40 Para limpar a compilação é necessário rodar o seguinte comando:
41 
42 ```
43 $ make clean
44 ```
45 
46 Para limpar o arquivo de cobertura de testes:
47 ```
48 $ make clean_coverage
49 ```
50 Para limpar todos os arquivos compilados juntamente com os de cobertura
51 ```
52 $ make clean_all
53 ```
